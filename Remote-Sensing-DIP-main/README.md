# Remote-Sensing-DIP
# Remote Sensing Image Processing using DIP

## 📌 Project Overview
This project implements a complete Digital Image Processing (DIP) pipeline on satellite images to extract meaningful geospatial information.

---

## 📂 Dataset
- Source: Kaggle (Remote Sensing Satellite Images)
- Contains images of:
  - Urban areas
  - Agricultural fields
  - Water bodies
  - Barren land

---

## ⚙️ Techniques Used

### 1. Pre-processing
- Image resizing (256x256)
- Format standardization
- Histogram analysis

### 2. Noise Removal
- Gaussian Filter
- Median Filter
- Bilateral Filter

### 3. Segmentation
- K-Means Clustering
- Otsu Thresholding

### 4. Compression
- JPEG Compression
- PNG Compression

### 5. Feature Extraction
- Canny Edge Detection
- CLAHE (Contrast Enhancement)

---

## 📊 Evaluation Metrics
- PSNR (Peak Signal-to-Noise Ratio)
- SSIM (Structural Similarity Index)

---

---

## 🚀 Results
- Bilateral filter preserved edges best
- K-Means gave better segmentation for complex images
- JPEG achieved higher compression, PNG preserved quality
- CLAHE improved contrast in low-light regions

---

## 🎯 Conclusion
This project demonstrates how DIP techniques can effectively process satellite images for analysis and interpretation.
